# Free Underground Tekno Radio

![logo](https://github.com/fabriziosalmi/freeundergroundtekno/blob/main/logo.png?raw=true)
# Listen now: [https://radio.free-tekno.com](https://radio.free-tekno.com)

Free Underground Tekno Radio is an online radio station based in Italy dedicated to free tekno music founded by the members of the Space Invaders Tekno Sound collective in the late 2008.
It streams music 24/7 focusing on tracks from free parties and sound systems. The main archive contains 200+ recordings made by artists across decades all around the world while performing at the [WebTek Streaming Teknival](https://github.com/fabriziosalmi/webtek/blob/master/README.md) editions. 

If you want to know more about the WebTek jump [here](https://github.com/fabriziosalmi/webtek) ;-)

- Web radio stack: [Azuracast](https://github.com/AzuraCast/AzuraCast), [GitHub Pages](https://docs.github.com/en/pages).
